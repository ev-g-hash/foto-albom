================================================================================
НАЗНАЧЕНИЕ
================================================================================

Цифровой фотоальбом для размещения и просмотра фотографий с возможностью 
администрирования. Персональный альбом для Нурмеева Дениса Рашитовича 
(год рождения: 1977).

================================================================================
ФУНКЦИОНАЛ
================================================================================

1. ПРОСМОТР ФОТОГРАФИЙ
   - Галерея с пагинацией (6 фото на странице)
   - Лайтбокс для увеличенного просмотра фотографий
   - Lazy loading изображений для оптимизации загрузки
   - Список всех фотографий в разделе "Содержание"

2. СИСТЕМА АВТОРИЗАЦИИ
   - Страница входа для администраторов
   - Проверка прав суперпользователя
   - Безопасная аутентификация через Django Auth

3. УПРАВЛЕНИЕ КОНТЕНТОМ (ТОЛЬКО ДЛЯ АДМИНИСТРАТОРОВ)
   - Загрузка фотографий с автоматическим сжатием
   - Редактирование названий фотографий
   - Редактирование описаний фотографий
   - Удаление фотографий с подтверждением

4. АДАПТИВНЫЙ ДИЗАЙН
   - Адаптация под мобильные устройства (до 480px)
   - Адаптация под планшеты (768px - 1023px)
   - Адаптация под десктопы (1024px+)
   - Разные фоновые изображения для разных разрешений

================================================================================
СТРУКТУРА ПРОЕКТА
================================================================================

Основные директории:
- /templates/          - HTML шаблоны
- /static/             - Статические файлы (CSS, JS, изображения)
- /fotos/              - Django приложение для фото
- /data/               - Директория для данных (production, Amvera)
- /media/              - Загруженные изображения и превью

================================================================================
СТРАНИЦЫ ПРИЛОЖЕНИЯ
================================================================================

1. greeting.html (Главная страница)
   - Поздравление с Днём Рождения
   - Имя владельца альбома: Нурмеев Денис Рашитович
   - Год рождения: 1977

3. detail.html (Детальный просмотр)

4. conclusion.html (Содержание)

================================================================================
МОДЕЛЬ ДАННЫХ
================================================================================

Модель Photo:
- id: integer (PK, auto)
- title: varchar(200) - название фотографии
- description: text - описание фотографии
- image: ImageField - изображение (сжатое до JPG)
- created_at: DateTimeField - дата создания
- thumbnail: ImageField (опционально) - превью изображения

Валидация:
- Максимальный размер файла: 20MB
- Минимальные размеры: 100x100px
- Максимальные размеры: 8000x8000px

================================================================================
ТЕХНОЛОГИИ И ЗАВИСИМОСТИ
================================================================================

Основные библиотеки:
- Django 6.0 - веб-фреймворк
- Pillow 12.0.0 - обработка изображений
- Gunicorn 21.2.0 - WSGI-сервер для production
- python-dotenv 1.2.1 - загрузка переменных окружения
- whitenoise - обслуживание статических файлов

================================================================================
РАЗВЁРТЫВАНИЕ
================================================================================

Локальная разработка:
1. Установить зависимости: pip install -r requirements.txt
2. Выполнить миграции: python manage.py migrate
3. Запустить сервер: python manage.py runserver

Docker:
1. Собрать образ: docker build -t foto-album .
2. Запустить контейнер: docker run -p 8000:8000 foto-album

Amvera (production):
1. Настроить переменные окружения
2. Данные сохраняются в /data/
3. Используется Gunicorn с 2 worker-ами
